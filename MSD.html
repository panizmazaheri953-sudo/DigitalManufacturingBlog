→ Recommendation:
25 Bambu P1S printers running 24/7 is the optimal configuration to meet the 50,000 phones/year target. This is equivalent to operating 75 printers on a single 8-hour shift — a capital saving of £50,000+ in printer hardware alone, but requires a robust remote monitoring infrastructure to manage unattended overnight operation (addressed via camera system and cloud dashboard).

⚙️ How the FESTO Line Actually Works — Pipelined Flow
Wrong assumption: One case enters S1, waits for all 6 stations to finish (99s), then the next case enters.

Reality: The FESTO line is a pipeline — a new case enters S1 every 16.5 seconds (99s ÷ 6 stations). While Case A is at S6, Case B is at S5, Case C at S4, and so on. This means a completed case exits S6 every 16.5 seconds, not every 99 seconds. The total line throughput is ~218 cases per hour — far above the 150s takt requirement.

Formula: Throughput time per case = Total cycle time ÷ Number of stations = 99s ÷ 6 = 16.5s per case

Why 2 per shift? Each operator can realistically manage 12–13 printers — collecting prints every 2.5h, cleaning beds, changing filament, starting new jobs, and visual QC. 25 printers requires 2 operators. IoT alarms and Bambu Cloud handle monitoring; engineers are alerted if anything fails overnight.
Operator Tasks Per Shift
Print collection — collect completed pairs from print bed every 2.5h (~3 batches per 8h shift)
Bed cleaning — clean and re-prepare build plate between runs to prevent adhesion failures
Filament management — monitor spool levels, change PLA rolls before depletion mid-print
Job loading — start next print job via Bambu Cloud; verify settings match locked production profile
Visual QC — inspect each print for warping, surface defects, incomplete layers
FESTO S3 iPick — manually place and press casings together at Station 3
Reject handling — log failed prints (20% rate), clear bed, restart

⚠ Bottleneck Recommendation
Station 5 (Oven) at 35s is the FESTO line bottleneck. However, as the total FESTO cycle (99s) remains well within the 150s takt, the bottleneck does not currently limit production. If demand scales beyond 50,000 pairs/year, a second FESTO oven module or parallel FESTO line would be the recommended investment.
